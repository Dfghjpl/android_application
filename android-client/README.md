# Android клиент для LocalMesh Chat

Это мобильное приложение для подключения к серверу LocalMesh.

## Возможности
- Подключение к локальному серверу
- Отправка и получение сообщений
- Геолокация участников
- SOS-сигнал

## Установка
1. Подключись к Wi-Fi сети LocalMesh
2. Запусти приложение
3. Введи IP адрес сервера
